How to run the tests:

- Method 1 - through the terminal
- Method 2 - through IntelliJ
```

Method 1 - through the terminal:
1. Open the terminal
2. Navigate to the root of the project folder
3. Run the command: "./mvnw test"
4. The tests will run and the results will be displayed in the terminal
```
Alternatively: 
```
Method 2 - through IntelliJ:
1. Open the project in IntelliJ

Then:
2. Right click the root folder of the project (can also right click on the test folder)
3. Select "Run all tests"

Or, to run select test files:
2. Navigate to the test folder

Then:
3. Right-click on any test file and select "Run *TestFileName*"
4. The tests will run and the results will be displayed in the "Run" tab

Or, to run individual tests:
3. Open the test file and click on the green arrow next to the test method
4. The test will run and the results will be displayed in the "Run" tab
```
